Информация о amoCRM рассылки

СМС-рассылки через amoCRM

Отправляйте СМС-рассылки не выходя из amoCRM, с помощью виджета P1SMS

Подключить Амо виджет

Шаги настройки интеграции СМС

  • Для подключения виджета зайдите в настройки amoCRM в разделе интеграции найдите виджет P1SMS
  • Заполните поля "Телефон", "Email", "Пароль"
  • Перейдите в любой контакт и создайте свой первый шаблон

Что умеет виджет p1sms в amoCRM

Автоматизировать воронку продаж Настраивайте работу виджета p1sms в Digital Pipeline, так чтобы при переходе в этапы вашей воронки, клиенту приходило уведомление по смс.

Отправлять массовые смс сообщения не выходя из amoCRM Выберите список контактов или сделок, которые необходимо проинформировать. С помощью виджета создайте текст сообщения смс. Выберите нужное имя отправителя и запускайте рассылку без модераций. Рассылка отправится вашим контактам моментально!

Отправлять сообщения прямо в Viber При настройке виджета в цифровой воронке или при отправке сообщений по списку контактов, выберите в настройках отправить в Viber. Текст сообщения становится больше, появляется возможность отправить медиафайлы вместе с текстом.

Запускайте СМС-рассылки в amoCRM уже сейчас!
