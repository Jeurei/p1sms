Информация о таргетной рассылке

Таргетированные СМС-рассылки

Таргетированная СМС-рассылка позволяет выбрать абонентов по полу, городу, возрасту, радиусу местонахождения, чтобы отправить сообщения тем, кому она будет максимально интересна.

Создать таргетированную рассылку

Выберите свой канал для рассылки с P1SMS

  • Способ привлечь новых клиентов через SMS-рассылки. Если у вас нет базы, или вам нужны новые клиенты, то с помощью этой рассылки вы сможете их найти.

  • Сделайте SMS-рассылку по тем клиентам, которые заинтересованны в ваших услугах. С помощью различных вариантов таргетинга вы сможете выбрать необходимых вам получателей.

  • Рассылка законна, потому что отправляется только тем абонентам, у которых подключена услуга «Мобильная реклама».

Для отправки таргетированной рассылки составьте портрет целевой аудитории из предложенных параметров. Чем больше параметров - тем дороже 1 SMS.

  • Местонахождение абонента: регион, область, город, район, улица
  • Возраст
  • Пол
  • Операционная система телефона: Android, Apple iOS, BlackBerry‎, Symbian‎, Windows Phone‎
  • Подключенные услуги у оператора: роуминг, интернет, тарифный план
  • Среднемесячные затраты абонента на сотовую связь
  • Технические характеристики мобильного телефона: производитель, модель
